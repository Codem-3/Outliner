Outliner - Privacy Policy

Last Updated: April 22, 2025

1. Introduction

Outliner ("we", "our", or "us") is a Chrome extension designed to help web developers inspect and outline page elements using customizable visual styles. We are committed to protecting your privacy, and this Privacy Policy outlines how the extension handles user data.

2. Information We Collect

Outliner does not collect, store, or transmit any personally identifiable information. The extension runs entirely locally in your browser and does not connect to external servers.

3. Local Storage

We use Chrome's local storage to save your outline preferences, including color, width, style, and element selectors. This information remains on your device and is never transmitted elsewhere.

4. Permissions

Outliner requires the following Chrome extension permissions to function properly:

  • activeTab: To access and interact with the current webpage only when the user enables it
  • scripting: To inject the outline and element info functionality into the page
  • storage: To store user settings and preferences locally

5. Third-Party Services

Outliner does not use any third-party services, analytics, or external content. All features work offline and securely within your browser environment.

6. Changes to This Policy

If any changes are made to this Privacy Policy in the future, they will be posted here with an updated "Last Updated" date.

7. Contact

For any questions or concerns about this Privacy Policy, please contact us via the Chrome Web Store listing or through the GitHub repository: https://github.com/Codem-3/Outliner
